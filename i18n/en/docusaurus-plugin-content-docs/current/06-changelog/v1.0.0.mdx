# 📦 Klleon Chat SDK v1.0.0

## ✨ Features

- Added `onStatusEvent` event
- Added `changeAvatar` method
- TypeScript support

## 🔁 Changed

- Renamed `stopEcho` to `stopSpeech` method

## 🛠️ Others

- Changed UI to WebComponent-based
  - Avatar: `KlleonChat.showStreaming()` → `<avatar-container />`
  - Chat: `KlleonChat.showChatUi()` → `<chat-container />`
