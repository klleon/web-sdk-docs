# Klleon Ondevice SDK

Klleon Ondevice SDK is a powerful SDK that provides on-device AI capabilities along with video processing and real-time messaging. With this SDK, you can easily integrate AI-based avatar chat functionality that communicates in real-time into your applications.

## Key Features

### Video Processing
- AI video playback
- LLM-based TTS
- High-performance on-device AI processing

### Real-time Messaging
- WebSocket-based real-time communication
- Message sending and echo functionality
- State management
