# Intro

# Klleon Ondevice SDK

Klleon Ondevice SDK는 비디오 처리, 실시간 메시징과 함께 온디바이스 AI 기능을 제공하는 강력한 SDK입니다. 이 SDK를 사용하면 AI 기반 아바타와 실시간으로 소통하는 채팅 기능을 애플리케이션에 쉽게 통합할 수 있습니다.

## 주요 기능

### 비디오 처리
- AI 비디오 재생
- LLM 기반의 TTS
- 고성능 온디바이스 AI 처리

### 실시간 메시징
- WebSocket 기반 실시간 통신
- 메시지 전송 및 에코 기능
- 상태 관리
