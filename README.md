# Klleon Chat SDK Documentation

Klleon Chat SDK 공식 문서 사이트.

- **URL**: https://docs.klleon.io
- **Stack**: Docusaurus 3.9.2 + React 19 + TypeScript
- **i18n**: 한국어(기본) / English / 日本語
- **SDK 버전**: 1.3.0

## Quick Start

```bash
# 의존성 설치
npm install

# 개발 서버 (한국어)
npm run start

# 특정 로케일
npm run start:en
npm run start:ja

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run serve
```

## Requirements

- Node.js >= 20.0
- npm (yarn 사용 금지)
